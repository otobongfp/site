Relay Protocol

Peer-to-peer communication protocol for decentralized messaging with secure, direct communication between nodes and built-in encryption. Built in Go for high performance and concurrency, the relay system enables low-latency message routing across distributed networks. Features include end-to-end encryption, message queuing, automatic node discovery, and support for various message formats including JSON and binary protocols.
