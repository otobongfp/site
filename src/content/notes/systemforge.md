#### Building for the command Line

SystemForge is a comprehensive DevOps toolbox built in Rust, designed to streamline system monitoring, file management, and security tasks for developers and system administrators. The goal is to provide a unified tool that integrates multiple essential features for maintaining and optimizing system health, file security, and performance.

Well I have barely kept to that bargain at the pace I earlier planned. So more recently I decided to get my hands even more dirty with rust by building a complete application toolbox.

My goal is that systeforge will have the following the features:
Custom Shell, File analyzer, Log File Analyzer, File Encryption Tool, System Monitor, Process Manager, Packet Analyzer, etc
